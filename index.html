Is Aerial Travel Becoming Safer?¶

Eric Hepp

Introduction¶

In this data science pipeline tutorial, we will collect, organize, and analyze data, gain and present insights from them. We will be looking at plane crash data to investigate whether aerial travel safety is improving or not. Data will be collected from planecrashinfo.com, which contains data of every civil and commercial plane crash where there were casualties since the Orville brothers took flight.

Before starting, we will need the following libraries:

Data Collection¶

We will be collecting data from http://www.planecrashinfo.com/database.htm. It contains data on fatal aviation accidents since 1908. Every accident contains detailed information, but each is on its own page, so the first task is to collect it into one single table. BeautifulSoup reads the data, allowing us to put it into a Pandas data frame, while Pandas data frames are comparable to SQL tables, where each entry represents a row of a table. We use Pandas here as Pandas tends to be better suited to converting data in many formats into a table.

#define a function to make requests at a throttled rate
@sleep_and_retry
@limits(calls=60, period=60) #one request per second on average, 5000 requests to make = 1.5 hours
def rqget(url):
    response = rq.get(url)
    if response.status_code != 200:
        raise Exception('API response: {}'.format(response.status_code))
    return response

#load the data
url_base = "http://www.planecrashinfo.com/"
all_frames = []
for year in range(1920,2022):
    #get the tables for each year
    response = rqget(url_base + str(year) + "/" + str(year) + ".htm")
    soup = bs(response.content, "html.parser")
    #get the links to individual crash data from the year tables
    href_tags = soup.find_all('a', href=True)
    for link in href_tags[:-1]: #skip the "return to home page" link at the end
        response2 = rqget(url_base + str(year) + "/" + link['href'])
        soup2 = bs(response2.content, "html.parser")
        frame = pd.read_html(response2.text, flavor="bs4")
        frame = frame[0][1:].transpose()
        frame.columns = frame.iloc[0] #set the first row as the frame header
        frame = frame[1:] #remove the header row
        all_frames.append(frame)
#combine all the rows
data = pd.concat(all_frames)

Tidying data¶

Next we need to convert the table into something where the data can be manipulated. For instance, the "Aboard" and "Fatalities" columns have data that is readable to , but it will take a bit of work to get them into numbers that can easily be used by the computer. At this stage we will also be replacing the missing data entry question marks with NaN values.

In [6]:
data.columns = [col[:-1] for col in list(data)] #drop the colons in the header
#convert dates to Datetime values
data["Date"] = [datetime.strptime(date, "%B %d, %Y") for date in data["Date"]]
#split aboard/fatalities, passengers/crew
pattern = re.compile(".*passengers:(\d+|\?) crew:(\d+|\?).*")
matches_aboard = [pattern.match(x) for x in data["Aboard"]]
matches_fatalities = [pattern.match(x) for x in data["Fatalities"]]
data["Passengers Aboard"] = [x.group(1) for x in matches_aboard]
data["Crew Aboard"] = [x.group(2) for x in matches_aboard]
data["Passenger Fatalities"] = [x.group(1) for x in matches_fatalities]
data["Crew Fatalities"] = [x.group(2) for x in matches_fatalities]
#drop summary, time, aboard, fatalities
data = data.drop(labels=["Summary","Time","Flight #","Aboard","Fatalities"],axis=1)
#rename "ground" to "ground fatalities"
data.columns = ["Ground Fatalities" if x == "Ground" else x for x in data.columns]
#replace ? values with NaN
for column in list(data):
    data[column] = data[column].replace("?", np.nan)
#convert string values to numeric values
data["Passengers Aboard"] = [int(x) if x == x else np.nan for x in data["Passengers Aboard"]]
data["Crew Aboard"] = [int(x) if x == x else np.nan for x in data["Crew Aboard"]]
data["Passenger Fatalities"] = [int(x) if x == x else np.nan for x in data["Passenger Fatalities"]]
data["Crew Fatalities"] = [int(x) if x == x else np.nan for x in data["Crew Fatalities"]]
data["Ground Fatalities"] = [int(x) if x == x else np.nan for x in data["Ground Fatalities"]]
#add column for total fatalities
data["Total Fatalities"] = data["Passenger Fatalities"] + data["Crew Fatalities"] + data["Ground Fatalities"]

#exploration/visualization
#line graph accidents/year
data = data[data["Total Fatalities"].notna()]
fatalities = {}
for i,row in data.iterrows():
    if row["Date"].year not in fatalities.keys():
        fatalities[row["Date"].year] = []
    fatalities[row["Date"].year].append(row["Total Fatalities"])
years = fatalities.keys()
accidents_per_year = [len(x) for x in fatalities.values()]
deaths_per_year = [sum(x) for x in fatalities.values()]
#plot fatalities per year
fig = plt.figure()
ax = plt.axes()
ax.plot(years, deaths_per_year)
plt.xlabel("Year")
plt.ylabel("Aviation Accident Fatalities")
plt.title("Aviaation Accident Fatalities per Year")
plt.show()

The spike in 2001 is from the 9/11 attacks, causing an uncharacteristic spike in casualties. Other than that, the trend appears to be going downward. The 2001 casualties were caused by a small number of attacks, so looking at the quantity of accidents should not be as affected.
